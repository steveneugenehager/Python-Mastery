URL = https://www.pypi.org

What is PyPI?
    - It is a central platform where developers publish, share, and distribute Python packages.
    - It hosts thousands of third-party modules and libraries that you can install and use in your Python projects.

Purpose:
    - PyPI makes it easy to find, install, and manage third-party Python software.
    - It helps foster a large ecosystem of reusable Python components.

How do you use PyPI?
    The most common way to access packages from PyPI is through the pip tool:

    pip install package_name
